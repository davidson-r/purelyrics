அவர் எந்தன் சங்கீதமானவர்<br/>
அரணான கோட்டையுமாம்<br/>
ஜீவனின் அதிபதியான அவரை<br/>
ஜீவிய காலமெல்லாம் வாழ்த்திடுவோம்<br/><br/>

துதிகளின் மத்தியினில் வாசம் செய்யும்<br/>
தூதர் கணங்கள் போற்றும் தேவன் அவரே<br/>
வேண்டிடும் பக்தர்களின் குறைகள் கேட்கும் ( 2)<br/>
திக்கற்ற பிள்ளைகளின் தேவன் அவரே ( 2)            - அவர்<br/><br/>

இரண்டு மூன்றுபேர் எந்தன் நாமத்தினால்<br/>
இருதயம் ஒருமித்தால் அவர் நடுவில்<br/>
இருப்பேன் என்றவர் நமது தேவன் ( 2)<br/>
இருகரம் தட்டி என்றும் வாழ்த்திடுவோம் ( 2)            - அவர்<br/><br/>

வானவர் கிறிஸ்தேசு நாமம் அதை<br/>
வாழ்நாள் முழுவதும் வாழ்த்திடுவோம்<br/>
வருகையில் அவரோடு இணைந்து என்றும் ( 2)<br/>
வணங்குவோம் வாழ்த்துவோம் போற்றிடுவோம் ( 2)     - அவர்
</center>
